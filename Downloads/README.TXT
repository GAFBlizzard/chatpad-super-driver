Hi, this is the README.TXT file for the Chatpad Super Driver by GAFBlizzard (A.K.A. Blizzard A.K.A. AgentElrond).

Please see the driver website (https://github.com/GAFBlizzard/chatpad-super-driver/) for information about the driver,
how to install and use it, and to get updated versions.




************************************
Lists of configuration file strings:
************************************

Here are some lists of strings that can be used in chatpad_config.txt.  See the
comments in chatpad_config.txt for information on how to use these strings.
These strings are not case-sensitive.

=======================================

   (Note that the left shift or left shift latch should normally be used rather
    than right shift, since right shift and right shift latch currently do not
    act as chatpad modifiers.  They also do not use the shift key indicator light.
    They may not even work properly.  Basically, ask GAFBlizzard if you need special
    functionality from the right shift key.)
